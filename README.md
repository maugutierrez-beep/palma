# GVC Capital · Inteligencia de inversión palmera

Plataforma web para evaluar inversiones en palma de aceite en Colombia (Zona Norte / Cesar).
Aplicación autocontenida en `index.html`. Publicada en GitHub Pages.

**En vivo:** https://maugutierrez-beep.github.io/palma/
**Clave de acceso:** `gvc2026`

## Editar y publicar
Edita `index.html` y haz push a `main`. GitHub Pages actualiza el sitio automáticamente.

```
git add index.html
git commit -m "tu cambio"
git push origin main
```
